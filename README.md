# Custom OS image based on Aurora

A custom bootc image based on [Universal Blue's Aurora](https://github.com/ublue-os/aurora).

Custom changes so far:
- OpenConnect in a pre-release version (https://copr.fedorainfracloud.org/coprs/dwmw2/openconnect/)


# Image
```bash
sudo bootc switch ghcr.io/mafuba8/aurora-hoh
```
